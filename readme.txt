== Description ==

Easy WP turns your wordpress-admin into a super-simple CMS. It allows your clients to easily edit there pages and content.
Next to that it can tap into the wordpress.com stats-plugin, so your clients can check there website stats themselves. 

Easy WP is the brainchild of collaboration between To Wonder multimedia and Motief:Collectief. We use it for every small
website, that needs a simple admin. If your interested in extending the simple CMS, please contact To Wonder at http://www.easy-wp.com


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `Easy-wp` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it! You can toggle between the simple and admin view with the button in the top-right corner.
